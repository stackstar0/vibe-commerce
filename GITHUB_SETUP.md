# üéØ Quick Start: Push to GitHub

## Step 1: Create GitHub Repository

1. Go to: https://github.com/new
2. **Repository name**: `vibe-commerce` (or your choice)
3. **Description**: "Premium Fashion E-Commerce Platform - Full Stack MERN"
4. **Visibility**: Public (recommended) or Private
5. **DO NOT** initialize with README, .gitignore, or license
6. Click "Create repository"

## Step 2: Push Your Code

After creating the repository, GitHub will show you commands. Use these:

```bash
# Add the remote (replace YOUR_USERNAME with your GitHub username)
git remote add origin https://github.com/YOUR_USERNAME/vibe-commerce.git

# Push your code
git push -u origin main
```

### Example (replace 'stackswift' with YOUR username):
```bash
git remote add origin https://github.com/stackswift/vibe-commerce.git
git push -u origin main
```

## Step 3: Verify Upload

1. Refresh your GitHub repository page
2. You should see all files uploaded
3. Check that README.md displays properly
